# Flash Français

Application d'apprentissage du français utilisant des flashcards.

## Structure du Projet

```
flash-francais/
├── frontend/           # Application React
├── backend/           # API Python
└── docker/            # Configuration Docker
```

## Prérequis

- Node.js >= 16
- Python >= 3.8
- PostgreSQL >= 13
- Docker (optionnel)
